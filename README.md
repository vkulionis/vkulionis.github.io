# useeio-analysis

Portfolio and experimental USEEIO analysis site built with the [Vercel Next.js Portfolio Starter Kit](https://vercel.com/templates/next.js/portfolio-starter-kit).

## Development

```bash
npm install
npm run dev
```

Open <http://localhost:3000>.

## Build

```bash
npm run build
```

The legacy static analysis pages are served from `public/`:

- `/network_full.html`
- `/linkages.html#linkages`
- `/flow_analysis.html`
